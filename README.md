# Job Search Console

A Python-based console application that scrapes job listings and lets you browse and apply for jobs directly from the terminal.

---

## What It Does

- Scrapes live job listings including job title, company, and location
- Browse all open positions in a clean, readable format
- View jobs by location or company
- Apply to any listing directly from the console

---

## Tech Stack

- Python
- BeautifulSoup4
- Requests

---

## How to Run

**1. Clone the repo**
```bash
git clone https://github.com/yourusername/WebScraper.git
cd WebScraper
```

**2. Install dependencies**
```bash
pip install requests beautifulsoup4
```

**3. Run the app**
```bash
python main.py
```

---

## Usage

Once running, you will see a menu in the console:

```
==================================================
           JOB SEARCH CONSOLE
==================================================
  1. Browse & Apply to Jobs
  2. View Locations
  3. View Open Positions
  4. View Companies
  5. Quit
==================================================
```

Select an option by entering the corresponding number. When applying, enter the job number and the application link will be displayed.

(Job postings are not real, and are only an example for this program)
